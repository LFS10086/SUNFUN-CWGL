# 三峰整装经销商客户整装财务收支管理系统本地版

本仓库保存本地运行版项目源码与桌面端构建配置。默认数据保存在当前电脑，适合客户在本机安装、试用和单机管理。

## 默认登录

- 经销商代码：`admin`
- 职位：`经销商`
- 密码：`123456`

## 已实现模块

- 客户档案、合同总价、签单总价、木门/定制订单号。
- 装修预算维护、预算识别、预算超支提醒。
- 收入收款、支出管理、工程人员、业务提成。
- 票据归档和统计报表。
- 本地经销商代码隔离、账户职位权限、密码修改。

## 本地运行

```powershell
npm install
npm run dev
```

默认访问：

```text
http://127.0.0.1:5173/
```

## 打包本地桌面端

```powershell
npm run package:win
```

输出目录在 `D:/workspace/sanfeng-finance-delivery/exe`。
